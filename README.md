# 📊 Sales Forecasting Dashboard

A full-stack sales analytics dashboard built using **React, Tailwind CSS, Recharts, and Flask**.

---

# 🚀 How to Run the Project

## 🔹 1. Start Backend

```bash
pip install -r requirements.txt
python server.py
```

👉 Runs on: http://127.0.0.1:5000

---

## 🔹 2. Start Frontend

```bash
cd frontend
npm install
npm run dev
```

👉 Runs on: http://localhost:5173

---

## ⚠️ Important

* Run **both backend and frontend**
* Backend must start first
* Python 3.10+ recommended

---

# ✨ Features

* 📈 Sales overview dashboard
* 📊 Trends & distribution charts
* 🔍 Search & filters (year, category, etc.)
* 📉 Sales forecasting using SARIMA
* 📋 Sales records table
